# Kenya Counties Analysis

Capstone project for Python Data Analysis. Looking at population, poverty, and development across Kenya's 47 counties.

## What its about
Trying to figure out which counties in Kenya are doing well and which ones are struggling, using actual data from the 2019 census and other public datasets.

## Data
All from HDX (https://data.humdata.org):
- Kenya 2019 Census (population, gender, density)
- GINI coefficient per county (income inequality)
- County poverty rates
- Human Development Index per county
- County boundaries GeoJSON for mapping

## Libraries
NumPy, Pandas, Matplotlib, Seaborn, GeoPandas
